Put your overlay art here for the desktop/game blocker.

Supported formats:
- .png
- .jpg / .jpeg
- .webp
- .bmp

How images are applied:
- The overlay agent auto-loads all images from this folder.
- Each blocked zone picks one image deterministically by zone id.
- Images are center-cropped to fit the zone (`UniformToFill`).

Tip:
- Add 2-5 images with distinct styles to quickly compare looks.
